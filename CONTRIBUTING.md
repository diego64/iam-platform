# Contribuição

1. Branch com prefixo válido: `feat/rbac`, `fix/login-timing`, etc.
2. Commits em Conventional Commits PT-BR — o hook `commit-msg` valida.
3. PR usando o template; gates do CI precisam estar verdes.
4. Endpoints novos exigem atualização de `openapi/openapi.yaml` e da collection Postman.
